**#TesloDb Api**

1. Clonar proyecto

2. Ejecutar npm install

3. Clonar el archivo `.env.template` y renombrarlo a `.env`

4. Cambiar las variables de entorno

5. Levantar la base de datos

```
docker compose up
```

6. Levantar el modo desarrollo

```
npm run start:dev
```

7. Ejecutar SEED

```
localhost:3000/api/seed

```
